# Vídeo da seção Home

Coloque aqui o arquivo de vídeo **Fertilizante Terravik**.

**Nome obrigatório:** `fertilizante-terravik.mp4`

Caminho completo: `public/video/fertilizante-terravik.mp4`

Se o arquivo tiver outro nome, edite `src/components/home/VideoSection.tsx` e altere a constante `VIDEO_SRC`.

**Nota:** Enquanto o vídeo não existir, a seção fica oculta automaticamente (sem área preta).
